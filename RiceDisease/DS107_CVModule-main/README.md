# DS107 CV Module

This module wraps the YOLO11s rice pest detector used by the RiceCare web app.

## Role

```text
uploaded image
-> YOLO11s detection
-> top detection becomes the pest-class prior
-> annotated image is cached
-> confidence IR is created
-> DS107 advisor session receives class_id + confidence + notes
```

The CV module does not generate agricultural advice. Advice is generated by the repository-root `advisor/` module through Gemini RAG.

## Model

Expected weight file:

```text
E08_yolo11s_img416_default_best.pt
```

Default inference settings:

```text
imgsz=416
conf=0.25
iou=0.70
max_det=20
device=cpu
```

The class list matches the paper, advisor, benchmark, and YOLO notebook:

```text
asiatic_rice_borer
brown_plant_hopper
paddy_stem_maggot
rice_gall_midge
rice_leaf_caterpillar
rice_leaf_hopper
rice_leaf_roller
rice_water_weevil
small_brown_plant_hopper
yellow_rice_borer
```

## Run One Image

From this folder:

```powershell
python -m cv_module.cli --image input_images\sample.jpg --output classification_results.json
```

Run a folder:

```powershell
python -m cv_module.cli --input-dir input_images --output classification_results.json
```

## Output Contract

Important fields:

```json
{
  "summary": {
    "has_detection": true,
    "predicted_label": "brown_plant_hopper",
    "predicted_label_vi": "ray nau",
    "predicted_confidence": 0.91,
    "count": 1
  },
  "annotated_image": "cache/bbox/example_bbox.jpg",
  "confidence_ir": {
    "level": "high",
    "needs_llm_confirmation": false,
    "llm_strategy": "direct_advice"
  },
  "ds107_payload": {
    "class_id": "brown_plant_hopper",
    "disease": "ray nau",
    "confidence": 0.91
  }
}
```

## Python API

```python
from cv_module.inference import classify_image

result = classify_image(
    image_path="input_images/sample.jpg",
    cache_upload=True,
    include_ds107_payload=True,
)
```
